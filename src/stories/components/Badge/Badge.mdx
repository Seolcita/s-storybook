## Overview

The Badge component is a small, versatile element used to display numbers, status, or tags. It's often used in conjunction with other components to indicate new messages, notifications, or additional information about an item. The Badge can be customized with different background colors using the bgColor prop. The component includes different variants such as Default and Text, which can be used in various contexts based on the user interface requirements.
