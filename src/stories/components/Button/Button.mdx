## Overview

The Button component is a fundamental interface element that triggers a function when clicked. It is customizable with various properties such as bgColor for background color, textColor for the color of the text, size for the size of the button, disabled to enable or disable the button, hasShadow to add a shadow effect, and borderRadius to adjust the roundness of the button's corners. The component includes different variants like Primary and Outlined, which can be used in different contexts based on the user interface requirements.
