## Overview

The Toast component is a small, non-blocking notification that appears to provide quick, at-a-glance feedback to the user. It can display different types of messages, such as success, error, or informational messages, based on the variant prop.<br />

#### The Toast component automatically disappears after 4seconds, except for error messages which require manual dismissal.

It includes an icon and a close
button for manual dismissal. The message prop is used to specify the text displayed
in the toast notification. This component is useful for providing feedback about
an operation's result or status.
