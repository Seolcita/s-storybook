## Overview

Typography component is used to maintain consistent styling for text throughout an application. It includes variants for different text sizes and styles, such as headings and body text. The Typography component can be customized with properties like variant to specify the text style, fontWeight to adjust the thickness of the text, and color to change the text color. This component is essential for maintaining a consistent look and feel across all text in an application.
